# Pattern Generator ⭐

A web app that generates star patterns of different shapes and sizes.

## Patterns Available
- Triangle ▲
- Pyramid △
- Diamond ◆
- Square ■
- Reverse Triangle ▽

## Features
- Choose pattern type from dropdown
- Choose any size from 1 to 15
- Animated pattern display
- Clean dark interface
- Originally built in Python, converted to a web app

## Technologies Used
- Python (original logic)
- HTML
- CSS
- JavaScript
